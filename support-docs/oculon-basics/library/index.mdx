# Library

## **Library:**

From Library you can access your collection of Integrations, Metrics and Dimensions

**How to connect or integrate?**

![](https://t9016375415.p.clickup-attachments.com/t9016375415/8f286fbe-58ce-4754-83af-719007f2d40a/Add%20Integration.png)

Go To Libraries and Click on “New integrations”

From the list of integrations select the required one

![](https://t9016375415.p.clickup-attachments.com/t9016375415/cb5b2dc9-a97a-4b1b-b2b8-f8d021d545a1/List%20of%20Integration.png)

**Authentication of the integration**

Once you are successfully connected an authentication pop-up will appear

![](https://t9016375415.p.clickup-attachments.com/t9016375415/5724c5fa-2010-4cee-b7c5-d0cd688cc39c/Authentication%20of%20Integration.png)

You can see your connected integration on the integration page.

**How to run a query and create database?**

Click on the Integration.

A space for writing query will appear

Start with the Query Name, Select Table from the dropdown and select the fields from the field dropdown.

[image]

Click on "Run Query", add or remove fields based on your requirements.

**Creating database:**

Once you are satisfied with the Query fields you can proceed to create a database out of it by just clicking on "Create Database"

Don't forget to give a proper name to your Database, Click Submit, your Database is ready

You can view your created Database under "Database" Tab in Library

**Creating a Database from Template:**

Click on "3-dot" icon and select "Create Database from Template" option

From the list of template chose the appropriate one to create the Database
