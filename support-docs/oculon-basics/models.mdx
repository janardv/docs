Models in Oculon form the foundation of financial planning and analysis. They allow users to build, customize, and analyze financial data, enabling better decision-making through scenario modeling, forecasting, and data-driven insights.

## **What you can do with models?**

1. Model Creation
2. Time Series Analysis
3. Scenario Planning & Comparison
4. Create Variables & Dimensions
5. Human redable Formulas
6. Collaborate
7. Integration with External Data

### **Model Creation**

You can create a new model from scratch. Getting start with a model is very simple, easy and intuitive in Oculon:

Go to Model Studio and click “Create New Model”.

Choose between Blank Model

Define key details like name, description, and data structure.

Creating Variables and Dimensions in the model is crucial for any model building. In Model Studio you can create these in a very organised and clean way. You can attach and detach dimensions with a variable or keep it independent based on the needs.

### **Collaborate**

We are building Oculon keeping collaboration at the core. We understand how sharing with the team members makes the work collaborative. You can share models with teammates using role-based access controls. Assign access level based on the role. Also you can track changes with version history and audit logs.

[image]

### **Versions**

Version is nothing but a state of a model or dashboard at a particular point of time. You can compare different model versions to track changes and evaluate impacts. If you have to undo just roll back to the previous versions.
