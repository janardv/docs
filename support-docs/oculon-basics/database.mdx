## **Creating a New Database:**

Oculon allows you to create, manage, and work with databases seamlessly within your financial models. You can connect external data sources, configure settings, and collaborate effectively.

![](https://t9016375415.p.clickup-attachments.com/t9016375415/3480f880-f30b-4b02-9b53-6e5c86ff82c5/Create%20Database.png)

o create a new database in Oculon:

1. Navigate to the **Database** section.
2. Click on **“New Database”**.
3. Enter a **name** and select the **data source**
4. Configure the **columns, permissions**. (You can add, delete)
5. Click **Create**, and your database is ready for use.

### **Other Actions:**

Connection Status:

You can check whether your database is connected and syncing properly.

- A **green indicator** means the connection is active.
- A **red indicator** means there’s an issue—click to troubleshoot.

**Configure Connection:**

To modify the database Configure the Connection:

- Go to the Configure Connection.
- Edit connection details
- Save changes to apply the configuration.

**View Data:**

To access the stored data and view:

1. Open the database.
2. Use options from Configure Connection to refine your view.

![](https://t9016375415.p.clickup-attachments.com/t9016375415/52b9373b-1a9b-48a3-ac98-8e4df663a703/View%20Data.png)

## **Grouping:**

You can group data within a specific field.

Click on the Group By option.

Select the rows to group data.

## **Create View:**

Steps to Create a View:

Open the database. (Configure Connection)

Apply filters as needed.

Click Save as View and give it a name.

Access saved views anytime for quick insights.

[image]

## **Share View:**

Steps to Share a View:

Open the view you want to share.

Click on **Share** and select the users or teams.

Set permissions (**View Only** or **Edit**).

Copy the link and share securely.
