Oculon supports a wide range of functions to help you build complex models with ease. Here’s a quick guide to the available functions:

**📐 Math Functions: Perform basic and advanced mathematical operations.**

SUM() — Adds values.

ROUND() — Rounds a number to a specified number of digits.

ABS() — Returns the absolute value.

POWER() — Raises a number to a power.

SQRT() — Returns the square root.

**⏱️ Time Functions: Work with dates and time periods in your models.**

TODAY() — Returns the current date.

YEAR() — Extracts the year from a date.

MONTH() — Extracts the month from a date.

DAYS() — Calculates the number of days between two dates.

EOMONTH() — Returns the last day of the month for a given date.

**💰 Financial Functions: Useful for financial modeling and analysis.**

NPV() — Calculates Net Present Value.

IRR() — Internal Rate of Return.

PMT() — Calculates loan payment amounts.

FV() — Future Value of an investment.

RATE() — Calculates the interest rate of an investment or loan.

**🎲 Probability & Statistical Functions: Analyze data with probability and statistical tools.**

AVERAGE() — Returns the mean.

MEDIAN() — Finds the middle value.

STDEV() — Calculates standard deviation.

COUNT() — Counts the number of values.

RANK() — Returns the rank of a number in a list.

**🔧 Other Useful Functions: Versatile functions for text manipulation, logical conditions, and more.**

IF() — Conditional logic.

AND(), OR() — Logical operators.

CONCATENATE() — Joins text strings.

LEN() — Returns the length of a string.

TEXT() — Converts numbers to formatted text.
