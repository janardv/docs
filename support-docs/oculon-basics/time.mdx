# Time

These are the tasks and actions can be performed with Time in Model, Dashboard, etc.

[image]

1. **Set the Date Range –** Define the start and end date of your model.
2. **Choose Time Granularity –** Select the level of detail for your model:

Daily

Weekly

Monthly (default & recommended)

Quarterly

Yearly

With the time granularity feature you can customize models for different reporting needs.

1. **The Time Filter allows you to:**

Adjust the time period displayed in your model and dashboard. Filter charts, tables, and reports to focus on specific time periods. Use Time Filter to adjust views based on actuals and forecasts.

[image]

1. **Setting Last Actual Date:**

The Last Actual Date determines how the latest available actuals in the model and how forecasts and actuals are separated in reports.

[image]

1. **Highlight Today**

To track the current time step ("Today") in your **model.** From settings just enable Today Line by toggling "Highlight Today" to visually mark the current period.
