## **Formatting a variable**

Formatting variables helps make your models clearer and easier to understand. You can apply various types of formatting depending on the kind of data your variable holds.

**Types of Variable Formatting**

1. **Numbers**: Format as decimals, percentages, or currency to match the context. For example:
    - **Revenue** → Currency (e.g., ₹, $, €).
    - **Profit Margin** → Percentage (e.g., 25%).
2. **Dates**: Use date formats to represent time-based data (e.g., YYYY-MM-DD or Month-Year).
3. **Text**: Some variables might hold descriptive data or categories, so keep this as plain text.
4. **Custom Units**: Add units like "kg," "hours," or "pieces" to represent specific quantities.
5. **Rounding**: Round to **2 decimal places** for cleaner metrics like 12.34% instead of 12.341256%.

## **Attaching Dimesnions in a Variable:**

Dimensions let you break down variables into more granular categories, such as regions, products, or departments. By attaching dimensions to a variable, you can analyze and model data at a detailed level.

**How to Attach Dimensions to a Variable**

1. Click Variable Editor
2. Select "Add Dimension"
3. Choose or Create a Dimension
    - Select from existing dimensions like **Region**, **Product**, or **Department**.
    - Or create a new dimension by defining categories (e.g., North, South, East for Region).
4. Apply the Dimension

## **Grouping:**

You can create a “**group**” from a set of variables. A nested group is also can be created inside a group. You just need to select the variables together and right click create group.

**Group Functionality**: Keep related variables together (e.g., group all revenue-related variables in one section).

## **Sections and Variables:**

Sections help you group and organize variables in your spreadsheet view for easy navigation and better structure.

## **Moving Variable:**

**Moving VariablesWithin the Same Model**

To move a variable inside your current model:

1. Drag and Drop: Just grab the variable and drag it to a different section.
2. Or Use the Menu: Click the three-dot menu next to the variable, select "Move," and choose the target section.

[image]

**Between Models**

Want to move variables across models? No need to recreate them from scratch!

1. Go to your target model.
2. Link It: Select the variable from the source model and link it in the new one (this way it stays updated automatically!).
3. Or Duplicate: If you prefer to keep a separate version, just copy and paste it manually.
