A short list of keyboard shortcuts for Oculon.

## Key notation

- ⌘ = Command (Mac) · Ctrl (Windows)
- ⌥ = Option (Mac) · Alt (Windows)
- ⇧ = Shift · ↵ = Enter/Return · ⎋ = Escape

### Global

- ⌘/Ctrl + K: Command Palette / Global search
- ⌘/Ctrl + P: Quick Open (recent/favorites)
- ⌘/Ctrl + /: Show shortcuts panel
- ⌘/Ctrl + F: Find in current view (Enter next, ⇧+Enter previous)
- ⌘/Ctrl + [ / ]: Back / Forward
- ⌘/Ctrl + 1…9: Jump to main areas (Home, Models, Databases, Dashboards, Feeds, Library, Agents, Integrations, Settings)

### Grids (Model Studio + Database tables)

- Enter: Edit focused cell · Esc: Cancel
- F2: Edit in place
- Tab / ⇧+Tab: Next/previous cell
- ⌘/Ctrl + C / X / V: Copy / Cut / Paste
- ⌘/Ctrl + ⇧ + V: Paste values only
- ⌘/Ctrl + Z / ⇧+Z: Undo / Redo
- Delete/Backspace: Clear contents
- ⌘/Ctrl + ↵: Fill selection with current entry
- ⌘/Ctrl + D / R: Fill down / right
- ⌘/Ctrl + ⇧ + =: Insert row/record (context-aware)
- ⌘/Ctrl + −: Delete row/record

### Filters + formulas (Model Studio)

- ⌘/Ctrl + ⇧ + L: Toggle filters
- ⌘/Ctrl + ⇧ + F: Filter builder
- F9: Recalculate selection
- ⌘/Ctrl + ⇧ + `: Show formulas

### Number formats (Model Studio)

- ⌘/Ctrl + ⇧ + 1…5: Number, Time, Date, Currency, Percent

### Database records

- E: Expand selected record
- Space: Preview attachments · ⌘/Ctrl + O: Open attachment

### Dashboards

- V: Toggle Builder mode

### Agents

- ⌘/Ctrl + ⇧ + A: Open Agent Console
- In console: ↵ run · ⇧+↵ new line

## Notes

We do not override browser/OS shortcuts like ⌘/Ctrl + N, W, Q, T.
